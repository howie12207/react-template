# React template

This is a react template.

## Structure

Vite + React + Recoil + Tailwind + Typescript

## Plugins

| Name          | Description                   |
| ------------- | ----------------------------- |
| [tailwind]    | A utility-first CSS framework |
| [fontawesome] | Icon library                  |
| [eslint]      | Check JS code                 |
| [typescript]  | Syntax for types              |
| [react-icons] | Icon for react                |
